# Tic Tac Toe Game 🎮

This is a simple Tic Tac Toe game built using HTML, CSS and JavaScript.

## Features
- Two player game
- Winner detection
- Draw detection
- Reset game button

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open the folder
3. Run the `index.html` file in your browser
